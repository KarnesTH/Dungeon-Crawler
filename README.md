# Dungeon Crawler C++

Console dungeon crawler written in C++.

## How to run

_Make sure you have a C++ Compiler installed_

Download or clone the source code

```bash
git clone https://github.com/KarnesTH/Dungeon-Crawler.git
```

Change directory

```bash
cd dungeon-crawler
```

Compile the code with `make`

```bash
make
```

Run the program

```bash
./main
```
